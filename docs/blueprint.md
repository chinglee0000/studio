# **App Name**: Twin3

## Core Features:

- User Authentication: Secure user authentication using Firebase Auth.
- Role Switching: Allows users to switch between 'Business' and 'User' roles with distinct interfaces.
- Quest Creation: Enables businesses to create quests with detailed configurations such as type, budget, and target audience.
- Quest Submission: Allows regular users to submit results for available quests.
- Data Visualization: Presents quest results through charts and graphs for easy understanding.
- Quest Approval and Payouts: Allows businesses to review and approve user submissions, triggering payouts.
- Create Quest Function: Cloud Function to handle quest creation, data validation, and Stripe payment processing.

## Style Guidelines:

- Primary color: Brand blue (#137FEC) serves as the seed color for Material Design 3 theming.
- Background color: Light gray (#F2F2F2) for light mode, dark gray (#212121) for dark mode.
- Accent color: A slightly darker shade of brand blue (#0E6DB3) for interactive elements.
- Body and headline font: 'PT Sans', sans-serif. Note: currently only Google Fonts are supported.
- Use Material Design icons throughout the application.
- Follow Google Material Design 3 specifications for component spacing and structure.
- Subtle animations for transitions between screens and loading states.
